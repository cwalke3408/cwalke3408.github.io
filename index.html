Visual Inspection Unit

More About Visaul Inspection Unit
This project, Visual Inspection Unit, was designed to have a collaborative team of Electrical and Computer Engineering students at Clemson University develop an autonomous inspection robot capable of identifying holes on a provided surface. Specifically used in this application to find holes in an airplane hull cut-out. Our team's design, affectionately named "Rover," is featured in this video and a synopsis is provided below.

Presented a point of reference to use as an origin and define an x-y plane, the unit traverses the surface in search of the holes. The identified holes are image captured, measured for diameter and then positioned on a map showcased on a Graphical User Interface (GUI). Each hole is logged in a file which is easily accessed on the user-friendly GUI. The GUI initializes two main functions for the surface probe, "Sweep" and "Inspect." Sweep finds the r covering the entire surface, while inspect is directed towards a specific point. The system is able to read in .csv files to compare measured values and locations against specification values and positions.

Mechanics and Hardware: Robot uses an independently powered 4WD Mecanum wheel system allowing for omni-directional movement, requiring 12-16 Vdc. Computer processing through a Raspberry Pi accompanied by Arduino and Alamode, and image capture through a Raspberry Pi camera module with serial input connection to the processor.

Four Ultra-Sonic sensors were used for edge detection. Two HP PS2 optical mice were used to assist in system odometry.
